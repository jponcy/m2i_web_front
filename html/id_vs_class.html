Les ID/class HTML

Utilité

Sert à pouvoir capturer/sélectioner des balises plus facillement.
Sert principallement pour le CSS/JS. Pour avoir des sélecteurs "simples".

Ca peut tout de même servir en HTML, par exemple :

  • les formulaires (lier un label à un champs)
  • liens d'ancrages

Principale différence

Un ID doit être unique dans la page HTML. Tandis qu'une classe peut-être réutilisé autant de fois que l'on veut.
