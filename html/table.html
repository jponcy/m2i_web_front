Tableau long

Pour les tableaux long (qui peuvent être imprimé sur plusieurs pages), nous devons rajouter de nouvelles balises : thead, tbody, tfoot. Elles permettent de bien séparer les titres/les pieds/contenu afin de pouvoir par exemple reproduire l'en-tête et le pied de tableau sur chaque page imprimé.
