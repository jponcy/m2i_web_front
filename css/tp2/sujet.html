Ajout d'un squelette

Personaliser le rendu du TP 1 afin d'y ajouter un squelette. Toutes les pages devront avoir ce squelette.

  1. avoir une bannière en haut de la page (une grosse image en largeur ; un fakeimg suffira)
  2. en dessous de cette bannière, on aura un menu horizontal avec les items suivants :
    • la page de liste de jeux
    • ajouter un nouveau jeu (TP 2 HTML)
    • à propos de nous (on mettra un peu de texte dans cette nouvelle page)
  3. le menu de navigation doit rester toujours visible sur l'écran (même quand on scroll très bas ; pas la bannière)
  4. un pied de page contenant un copyright (©)
