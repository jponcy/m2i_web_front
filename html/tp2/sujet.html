TP 2 - Gestion des formulaires

Faire un écran de formulaire permettant de saisir les informations concernant un jeu. Aucun traitement ne sera fait, nous ne voulons ici que le visuel du formulaire.
Nous rappelons que les informations que l'on peut saisir sont les suivantes :

  1. nom du jeu
  2. prix du jeu (ne peut pas être un négatif)
  3. une note moyenne de satisfaction sur 10 (entre 1 et 10 ; 0 n'est pas autorisé)
  4. une description détaillé (textarea)
  5. le genre de jeu (checkbox)
  6. le nom du dev (select)
  7. le nom de l'éditeur
