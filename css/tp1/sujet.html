Sujet du TP 1 de CSS

Reprendre le travail réalisé pour le premier TP d'HTML et le styliser avec les consignes suivantes :

  1. rajouter des bordures au tableau de la liste
  2. afficher les liens en orange
  3. changer la police d'écriture par défaut de toutes les balises au profit de 'Trebuchet MS'
  4. dans la vue de liste, sans rajouter d'HTML, mettre le nom des jeux en gras (font-weight)
  5. si ce n'est pas déjà fait, extraire tous les style (le CSS) dans un fichier dédié (d'extension .css)
  6. faire en sorte que les liens ne soient plus soulignés
  7. faire en sorte qu'un lien :visited ai le même visuel que les liens de base
