Sujet du TP 1

Nous souhaitons avoir 2 écrans :

  1. une liste de jeux vidéos ; on affichera les éléments suivants :
    1. nom du jeu
    2. prix du jeu
    3. une note moyenne de satisfaction sur 10
    4. un lien vers la fiche détaillé
  2. une fiche détaillé d'un jeu ; on affichera
    1. nom du jeu
    2. prix du jeu
    3. une note moyenne de satisfaction sur 10
    4. une description détaillé
    5. le genre de jeu
    6. le nom du dev
    7. le nom de l'éditeur
